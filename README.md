# Defensive Intelligence

**Transformer-Based Driver Behaviour Anomaly Detection 
for Kenyan Road Safety**

A physics-first implementation of Transformer 
architecture applied to dangerous driving detection 
on Kenyan road networks.

## Author: Benard Ogega
Mechanical Engineer | TVET Graduate & Lecturer | 
Road Safety Instructor — AA Kenya (FIA Affiliate)

## Project Goal
Detect dangerous driving patterns in real time using 
self-attention mechanisms trained on vehicle dynamics 
signals — speed, acceleration, brake pressure, 
and steering angle.
